# ASO Metadata Generator

A tool for generating App Store and Google Play metadata: titles, subtitles, keywords, and short descriptions — with live character counters.

Built with Claude Code.

**Note:** This is a template-based version. A full AI-powered version with Anthropic API generates metadata based on actual app description.
